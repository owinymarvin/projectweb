Application Developers

The car towing service app was created to enable stranded drivers tow their vehicles to the nearest or a preferred car garage. Users can request for a tow car, view the location of the tow car and pay for tow services all through the app.

Contact the development team

The above team was responsible for the design and implementation of the project. This was done in partial fulfillment of a bachelor of science degree in computer science.

Platform Suport

The car towing service was developed using flutter. Since flutter utilises one code base to build an application on various platforms, the "car towing application" can be utilised on android, ios, web, macos, windows and linux platforms.

More information about flutter and its one code base can be obtained from here
